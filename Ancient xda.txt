Flashing Instructions[/COLOR][/U][/B][/SIZE]

[SIZE="5"][color="#c3a13f"][B]CLEAN FLASH[/B][/SIZE][/COLOR]
[COLOR="red"]Make Sure you are Flashing it over Realme UI Firmware/ Don't flash it over Color os Firmware!![/COLOR]
- Make sure you're running the latest TWRP/OrangeFox
- Boot into recovery
- Wipe System, vendor, data, cache and dalvik cache 
- Flash  ROM.zip
- Format data
- Reboot to recovery (optional)
- Flash Magisk.zip (optional)
- Reboot to system
[SIZE="5"][color="#c3a13f"][B]DIRTY FLASH[/B][/SIZE][/COLOR]
- Boot into recovery
- Wipe cache and dalvik cache only
- Flash  ROM.zip
- Reboot to recovery (optional)
- Flash Magisk.zip (optional)
- Reboot to system
[b] First boot will take few minutes! Have patience! 
Clean Flash is always recommended![/b]
